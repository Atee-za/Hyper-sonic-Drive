PROJECT TITLE: Hyper-sonic Drive (car game)
PURPOSE OF PROJECT: To create a game that will end the boredom of all ages. Whether you old or young this game will make you feel happy.
VERSION or DATE: 2 November 2018
HOW TO START THIS PROJECT: Open the first World of this game 'MyWorld' and read the instructions then click play.
AUTHORS: Athenkosi Zono, Andile Mthethwa, Asiphe Mazamisa
USER INSTRUCTIONS:

 *Use arrows Left, Right, Up and down to control your car.
 *Try to avoid bumping into the blue cars coming towards you.
 *To proceed to next stage reach atleast score 45.
 *In Stage 2 and 3 you can bump into the ambulance to gain +1 life.
 *Try not to kill the pedestrains and animals crossing
The aim for the player is to finish stage 3 which is very dificult. But it is possible if you know how to drive.
